🚀 3D FlipBook Pro - WordPress Plugin Demo

Welcome to 3D FlipBook Pro!

This powerful WordPress plugin allows you to create stunning interactive 3D flipbooks from PDFs, images, and HTML content. Experience realistic page-turning animations powered by Three.js technology.

📚 How to Use

  1. Create a FlipBook: Go to FlipBook Pro → Add New in your WordPress admin
  2. Upload Content: Upload a PDF, multiple images, or enter HTML content
  3. Configure Settings: Set dimensions, controls, and other options
  4. Embed: Use shortcode or Gutenberg block to display your flipbook

🎯 Shortcode Examples

Basic Usage:

[flipbook id="1"]

Custom Dimensions:

[flipbook id="1" width="1000" height="700"]

With Auto-play:

[flipbook id="1" autoplay="true"]

Lightbox Mode:

[flipbook id="1" lightbox="true"]

No Controls:

[flipbook id="1" controls="false"]

🎮 Controls

  • Navigation: Click left/right sides of the book or use arrow buttons
  • Zoom: Mouse wheel or zoom buttons
  • Drag: Click and drag to rotate the book
  • Fullscreen: Click fullscreen button for immersive reading
  • Keyboard: Left/Right arrow keys for navigation, Escape to exit fullscreen

🛠️ Technical Specifications

  • WordPress: 5.0 or higher
  • PHP: 7.4 or higher
  • Browsers: Chrome 60+, Firefox 55+, Safari 12+, Edge 79+
  • Libraries: Three.js for 3D rendering, PDF.js for PDF support
  • Mobile: iOS Safari, Chrome Mobile, responsive design
